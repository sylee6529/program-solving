### 03-2. 계수정렬

- 주어진 배열의 **값의 범위가 작은 경우** 빠른 속도를 갖는 정렬 알고리즘. 출력 시 최대값만큼 반복문이 돌기 때문에, 범위가 작을수록 빠르다는 말이다.
- 1. count 배열을 만들고 -> 입력 원소(i)가 나올때마다 `count[i]++;` 한다. -> count 배열을 돌면서 값만큼 index값을 출력한다. (같은 작은 수부터 출력 완료)

<img width="813" height="362" alt="image" src="https://github.com/user-attachments/assets/00e101f4-c70f-4a0c-9c98-492a794c78b2" />


```java

import java.io.*;
import java.util.StringTokenizer;
import java.util.Arrays;

// 계수정렬
public class Main
{
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
		
		int N = Integer.parseInt(br.readLine());
		int[] count = new int[1001];
		StringTokenizer st = new StringTokenizer(br.readLine());
		for(int i=0; i<N; i++) {
		    int number = Integer.parseInt(st.nextToken());
		    count[number]++;
		}
		
		System.out.println(Arrays.toString(count));
		
		br.close();
		for(int i=0; i<=1000; i++) {
		    if(count[i] != 0) {
		        for(int j=0; j<count[i]; j++) {
		            bw.write(i+ " ");
		        }
		    }
		}
		
		bw.flush();
		bw.close();
	}
```

### 03-5. 다중 조건 정렬

- 다중 조건으로 정렬하는 경우 Comparable, Comparator 인터페이스 사용하기

**1) Comparable 인터페이스 사용하기**

```java
import java.io.*;
import java.util.Collections;
import java.util.ArrayList;

class Score implements Comparable<Score> {
    int english;
    int math;
    
    public Score(int english, int math) {
        this.english = english;
        this.math = math;
    }
    
    @Override
    public String toString() {
        return "Score(" + "eng=" + english + ", math=" + math + ")";
    }
    
    @Override
    public int compareTo(Score o) {
        // 영어를 우선, 그 다음은 수학기준
        if(this.english == o.english) return o.math - this.math;  // 파라미터로 들어오는 값 - this 값 하면 됨
        return o.english - this.english;
    }
}
```

```java
public class Main {
	public static void main(String[] args) throws IOException {
	    ArrayList<Score> myarr = new ArrayList<>();
	    myarr.add(new Score(80, 100));
	    myarr.add(new Score(100, 50));
	    myarr.add(new Score(70, 100));
	    myarr.add(new Score(80, 90));
	    
	    Collections.sort(myarr);
	    for(Score s : myarr) {
	        System.out.println(s.toString());
	    }
	    
	}
}
```
